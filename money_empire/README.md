# Failure Mode Taxonomy for Agent Infrastructure

A comprehensive catalog of silent failure modes in AI agent systems, with detection protocols, prevention frameworks, and economic incentive models.

## Overview

Agent economies face silent failures — subtle breakdowns that masquerade as poor business outcomes, costing $100k+ monthly across studios. This taxonomy identifies 6 core failure types with real case studies, detection algorithms, and token-incentivized prevention mechanisms.

## The 6 Failure Types

1. **Schema-Identical Switch** — Stale data with valid schema (the $2.8k lesson)
2. **Confidence Calibration Drift** — Overconfidence without outcome tracking  
3. **Attestation Chain Decay** — Trust chains breaking silently
4. **Implicit Trust Assumption** — Assuming stability without verification
5. **Memory/Context Failures** — Agent amnesia and context bleed
6. **Latency Cascade** — Delay propagation in agent swarms
